## var let const

#### var

这是ES5的语法，以前我一直都是用的这个，但其实它的定义域是function scope.

举个例子：

```javascript
function go(){
  console.log(a); //undefined
  var a = 100;
  console.log(a);//100
}
//上面代码其实相当于,这个机制好像叫做hoisting
function go(){
  var a, b;
  ...
}
//也相当于
function go(){
  let a;
  console.log(a); //undefined
   a = 100;
   console.log(a);//100
 }
```

再举个例子：

```javascript
var i = 100;
function a(){
	var i = 50;
  	console.log(i);//50
}

a();
console.log(i);//100
//这说明js会先查找小的scope再去查找大的scope，而且不会重复的赋值
```
